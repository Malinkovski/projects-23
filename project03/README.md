# Running the App:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

# Running the Database Server:

1. Start the database server:
   ```bash
   npm run server
   ```
# Running the sass:

1. Start the database server:
   ```bash
   npm run watch:scss
   ```
